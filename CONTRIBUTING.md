# 🤝 Contribution au projet UTL

Merci de votre intérêt pour contribuer au projet UTL!

## 📋 Guide de contribution

### 1. Fork le repository
```bash
fork GhassenGmatii/UTL
```

### 2. Clone votre fork
```bash
git clone https://github.com/YOUR_USERNAME/UTL.git
cd UTL
```

### 3. Créez une branche
```bash
git checkout -b feature/nom-de-la-feature
```

### 4. Faites vos changements
- Écrivez du code propre et commenté
- Suivez le style PEP 8
- Testez vos changements

### 5. Committez vos changements
```bash
git add .
git commit -m "Description claire de vos changements"
```

### 6. Poussez vers votre fork
```bash
git push origin feature/nom-de-la-feature
```

### 7. Ouvrez une Pull Request
- Décrivez clairement vos changements
- Expliquez pourquoi cette contribution est utile

## 📝 Code Style

- Utilisez PEP 8 pour le style Python
- Commentez votre code en français ou anglais
- Utilisez des noms de variables explicites

## 🐛 Signaler des bugs

Utilisez GitHub Issues pour signaler des bugs.

## 💡 Suggestions

Vos suggestions sont bienvenues! Ouvrez une issue pour discuter.

---

**Merci de votre contribution!** ❤️
